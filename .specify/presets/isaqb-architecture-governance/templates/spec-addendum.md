## Architecture Applicability

- Identify whether this feature affects system context, interfaces,
  building blocks, runtime behavior, deployment, quality attributes, or
  technical debt.
- List architecture goals and constraints relevant to the feature.
- Record quality attributes as concrete scenarios where they materially
  affect acceptance or design.
- Record whether architecture evidence under `docs/architecture/` is
  required.
- Record whether an ADR is required for any architecturally significant
  choice.
- Record whether security-relevant architecture is affected. If yes,
  apply the secure-architecture preset as well.
